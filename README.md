Car Showroom Management System (Java + MySQL)

Overview:
The Car Showroom Management System is a simple Java-based console application that connects to a MySQL database using the MySQL Connector/J library. It allows users to create accounts, log in, view available cars, and place orders â€” simulating a digital car showroom experience.

Technologies Used:
Java (JDBC) for backend logic and database connectivity
MySQL for data storage and management
mysql-connector-j-9.5.0.jar for connecting Java with MySQL

Features:
User registration (with name, contact, and state)
Secure login system
View available cars from database
Place car orders linked to user accounts
